---
name: c-ready
description: Show work items ready to be worked on, sorted by priority. Use when deciding what to work on next.
---

Call the `mcp__collective__collective_whats_ready` MCP tool to list work items that are:
- Status: READY (not blocked)
- Sorted by priority (high → low)
- With blocker information

Use this when:
- Starting a new session and need to pick work
- Finished current task and looking for next item
- Want to see what's unblocked and ready to go
