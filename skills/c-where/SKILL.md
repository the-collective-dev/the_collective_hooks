---
name: c-where
description: Show current work context - objective, phase, plan, and linked artifacts. Use when starting a session or after context loss.
---

Call the `mcp__collective__collective_where_am_i` MCP tool to show the current work context.

Display the results in a clear format showing:
1. Current objective (what you're working on)
2. Current phase (progress within the plan)
3. Active plan (the implementation plan)
4. Linked artifacts (ADRs, requirements)
5. Ready count and any blockers

This is the first thing to check when starting work or resuming after a break.
