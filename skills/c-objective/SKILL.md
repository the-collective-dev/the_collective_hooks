---
name: c-objective
description: Set the active objective for this session. Use before starting work to track what you're doing.
---

Set the active objective using the `mcp__collective__collective_set_objective` MCP tool.

Usage: /c-objective "Your task description"

This tells The Collective what you're working on, enabling:
- Better session tracking
- Accurate analytics
- Context preservation across sessions
- Work item linkage

Always set an objective at the start of your work session!
